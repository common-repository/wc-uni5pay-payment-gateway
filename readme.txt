== Description ==
This is a Uni5Pay+ Payment Gateway plugin for WooCommerce. This plugin uses the Uni5Pay+ Payment API (https://payment.uni5pay.sr) to redirect your customers to the payment page where they can scan the QrCode on there browser or click on "Pay with Uni5Pay+" in there mobile device. After successful payment the customer will be automaticly redirected back to the Woocommerce Order Received page. In the case of a cancelation the customer will be automaticly redirected back to the shopping cart. The Uni5Pay+ Payment API does not record or store any customer information.

Uni5Pay+ is a secure and easy mobile payment solution for users in Suriname for making local, international, and e-commerce payments.

== Installation ==
= Minimum Requirements =
* WordPress 6.2 or greater
* Woocommerce 8.3 or greater

= Automatic installation =
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "Uni5Pay+ Payment Gateway for Woocommerce" and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =
The manual installation method involves downloading our plugin and uploading it to your webserver via your favorite FTP application. The
WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
